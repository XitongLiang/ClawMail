## ClawMail 开发路线图

---

### Phase 0：基础架构搭建

**目标**：可运行的最小骨架，验证技术选型

| 模块 | 交付内容 |
|-----|---------|
| 核心框架 | 插件系统接口、事件总线、配置管理 |
| 数据层 | SQLite数据库初始化（WAL模式）、基础实体模型、FTS5虚拟表 |
| IMAP连接 | 163邮箱认证（aioimaplib）、收件箱列表获取 |
| 凭据安全 | keyring集成、credentials_encrypted加密存储与解密验证 |
| 最小UI | PyQt6 极简窗口（QListWidget 展示邮件主题，无样式）|

**里程碑**：成功登录邮箱，PyQt6 窗口显示邮件主题列表，凭据加密存储正常

**技术验证**：
- aioimaplib + qasync 异步 IMAP 连接稳定性
- OpenClaw 本地连接测试（openai SDK）
- keyring 在 macOS/Windows/Linux 的凭据存储
- 数据库 Schema 合理性（含 FTS5 触发器）

---

### Phase 1：邮件基础功能

**目标**：完整的基础邮箱体验

| 模块 | 交付内容 |
|-----|---------|
| 邮件同步 | 增量下载、附件保存、时间戳管理 |
| 邮件阅读 | 正文渲染（HTML/纯文本）、附件打开 |
| 邮件发送 | SMTP发送、草稿保存、回复/转发 |
| 文件夹管理 | 收件箱/草稿/已发送/回收站切换 |
| 基础UI | 三栏布局雏形（文件夹-列表-内容） |

**里程碑**：日常邮件收发无阻碍，替代Web邮箱基础使用

**暂不实现**：搜索、多账户、联系人管理

---

### Phase 2：AI核心能力

**目标**：统一AI处理流水线跑通

| 模块 | 交付内容 |
|-----|---------|
| OpenClaw集成 | 连接管理、流式响应、错误重试（3次指数退避） |
| 统一提取Prompt | 关键词+摘要+大纲+分类+任务 单次调用（Prompt #1） |
| AI离线降级 | ai_status=failed/skipped，UI显示⚠️徽章，自动加入重试队列 |
| AI处理流水线 | 新邮件自动触发、进度追踪、Qt Signal 通知 UI 更新 |
| 邮件摘要 | 列表显示 one_line 摘要、详情显示 brief + key_points |
| 智能分类 | 6个系统固定标签+动态项目标签、左侧分类栏可点击筛选 |

**里程碑**：新邮件到达后30秒内完成AI分析，UI实时更新（AI离线时立即显示原始邮件）

**关键优化**：
- 长邮件截断策略（超过 token 限制时截断 body_text）
- Token 消耗监控
- 结果缓存避免重复调用
- AI JSON 输出校验与默认值降级（见 prompt.md）

---

### Phase 3：ToDo系统

**目标**：任务管理闭环

| 模块 | 交付内容 |
|-----|---------|
| 任务提取 | AI自动识别行动项、截止日期推断 |
| 任务存储 | 完整Task表结构、状态流转 |
| 右侧面板 | ToDo列表UI、今日/未来/已完成分组 |
| 手动创建 | 快速输入框、NLP解析自然语言 |
| 邮件关联 | 点击任务跳回原邮件、完成任务标记邮件 |

**里程碑**：用户从"看到邮件"到"创建任务"到"完成归档"流畅无阻

**交互亮点**：
- AI提取的任务需用户确认（待确认分组）
- 语音/快捷方式创建任务
- 任务到期桌面提醒

---

### Phase 4：AI助手与撰写

**目标**：右下AI对话框和撰写辅助

| 模块 | 交付内容 |
|-----|---------|
| AI对话 | 上下文感知（当前邮件）、历史记录 |
| 智能回复 | 3种风格建议、一键插入编辑器 |
| 邮件润色 | 选中文字优化、语法检查、语气调整 |
| 智能撰写 | 从需求生成完整邮件、模板选择 |
| 撰写窗口 | 独立写信界面、富文本编辑、附件添加 |

**里程碑**：80%的邮件可通过AI辅助完成，无需从零撰写

---

### Phase 5：搜索与组织

**目标**：海量邮件可管理

| 模块 | 交付内容 |
|-----|---------|
| 全文搜索 | FTS5 索引（已在 Phase 0 建表），标题/正文/摘要/关键词检索 |
| 语义搜索 | ChromaDB 向量嵌入、自然语言查询（"找上周关于合同的邮件"），详见 tech_spec.md 第7节 |
| 高级筛选 | 日期范围、发件人、标签组合 |
| 搜索历史 | 保存常用查询、智能建议 |
| 邮件线程 | 同主题邮件聚合、对话脉络展示 |

**里程碑**：10万封邮件中5秒内找到目标

---

### Phase 6： polish与扩展

**目标**：产品级体验，个人工作流中心

| 模块 | 交付内容 |
|-----|---------|
| 日历集成 | 任务同步Apple Calendar、到期提醒 |
| 数据导出 | 备份/恢复、MBOX导出、迁移工具 |
| 多账户 | Gmail/Outlook/企业邮箱支持 |
| 移动端 | 基础手机查看、任务同步 |
| 插件市场 | 第三方Claw插件接口开放 |
| 协作功能 | 任务委派、共享项目空间 |
